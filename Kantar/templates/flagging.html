Spot Editing Guidelines


Flagging

Flag:

the tag applied to a spot indicating the spot’s origin as Network, Syndicated, Hispanic, Cable, or Local. It is used as part of the process for calculating rates and revenue. Each Network, Syndicated, Hispanic, or Cable title is attached to a specific set of spots per date of broadcast that must be flagged correctly.


Types of Flag Indicators:

Network:

Network spots are flagged blue and are indicated in the local edit with an ‘N’ in the origin column. Network spots are identified in the Network edits


  • Only the spots that match spots in the Network edit can be flagged.
  • A Network title can only be in a local edit with the correct set of Network spots.
  • If the Network title and spots in a local edit do not match the Network title and spots in the title’s Network edit, the title in the local edit needs to be moved, changed, or deleted.
  • Do not leave a Network title in a local edit without the correct set of flagged spots.

Syndication:

Syndicated spots are flagged red and are indicated in the local edit with an ‘S’ in the origin column. Syndicated spots are identified in the Syndication lists


  • A Syndicated title can only be in a local edit with the correct set of Syndicated spots.
  • If the Syndicated title and spots in a local edit do not match the title and spots provided in the title’s Syndication edits, the title in the local edit needs to be moved, changed, or deleted.*
  • *exception: Weekend syndication titles can match syndicated spots from Monday-Friday episodes.
  • Do not leave a Syndicated title in a local edit without the correct set of flagged spots

Hispanic:

Hispanic spots are flagged purple and are indicated in the local edit with an ‘H’ in the origin column. Hispanic spots are identified in the Spanish Network edits


  • A Hispanic title can only be in a local edit with the correct set of Hispanic Network spots.
  • If the Hispanic title and spots in a local edit do not match the Hispanic title and spots in the Hispanic edit, the title in the local edit needs to be moved, changed, or deleted.
  • Do not leave Hispanic titles in a local edit without the correct set of flagged spots.

Cable:

Cable Network spots are flagged green and are indicated in the local edit with a ‘C’ in the origin column


  • Cable Network spots are identified in the Cable Network edit
  • The same flagging rules apply as with Net, Syndication, and Hispanic titles:
  • Cable Network titles can only be in a local edit with the correct set of flagged spots

Local:

Local spots have no flagging indicator and no designator in the origin column. Local spots can air in any title.


  • Spots of different origins cannot be in the same title unless with local spots
  • (for example, Network & Local spots can be in the same title, but Network and Syndicated spots cannot be in same title. There should never be more than one color of flag in any title (blue, red, purple, OR green as indicated above)).
  • Local spots can occur with any other one type of spots.
  • Spots should not be flagged unless indicated in the corresponding logs. Do not flag random spots to fit a title in a local edit. Be sure the title and spots correctly match what is provided in the logs.

Flagging multiple spots:

  • Check boxes in local edit for each spot to be flagged/unflagged
  • Use the 'Global' icon to set origin

** Please remember flagging is important and it is the editor’s responsibility to make sure all titles in local edits are flagged correctly and accurately. **
