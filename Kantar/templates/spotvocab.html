Vocabulary


  • Market:

    The specific city in the U.S. or Canada that is monitored by Kantar Media. There are currently 215 markets monitored in Spot editing.

  • Station:

    Within each market, the stations are the individual networks monitored in that market. Some stations are National Network-affiliated (ABC, CBS, CW, FOX, NBC, ION, MNTV, TEL) or Independent (IND) which are not affiliated with any specific Network.

  • Broadcast Date:

    The Broadcast Date of an edit or episode title is when the edit or episode aired on television. Not to be confused with a calendar date. For example, on calendar date of Monday 6/19, Spot editors are working on Broadcast Date 6/14.

  • Local Edit:

    The local edit is the list of program titles and spots airing on a market's station for a specific broadcast date. The local edit is what Spot editors review and edit for accuracy.

  • Network Edit (a.k.a. Net Edit or Net Log):

    The Network edit provides the list of an individual Network's program titles, program lengths, and Network spots for a specific broadcast date. It is created by the Network editors after reviewing the Network video. Network titles can only air on that Network's local station or affiliate. Current Networks monitored in Spot include ABC, CBS, FOX, NBC, CW, MNTV, and ION.

  • Pacific Network Edit/Log:

    Pacific Network Edits provide information for West Coast/Arizona/Alaska/ Hawaii programming for a specific broadcast date. Pacific edits are only referenced when needed. Current Pacific Network feeds include ABCP, CBSP and NBCP.

  • Cable Network Edit/Log:

    The Cable Network edit provides the list of an individual Cable Network's programming information airing on a specific Cable Network for a specific broadcast date.

  • Syndication Edit (a.k.a. Synd Edit/Log):

    The Syndication edit provides the list of specific Syndicated spots that are airing with a Syndicated title as well as the length of the Syndicated title. Syndicated titles can air on any Network's local station.

  • Hispanic Edit:

    Hispanic edits provide the list of an individual Hispanic Network's program titles, program lengths, and Hispanic Network spots for a specific broadcast date. It is created by the Hispanic Network editors after reviewing the Network video. Current Hispanic Networks inclue AZTA, TEL, UMA, and UNI.

  • Spot:

    Spot is another term for 'commercial' or 'ad'. Types of spots are 'Brand'; 'Promo'; 'Paid Promo'; and 'PSA'

  • Brand Spots:

    Brand spots include stores and individual products (Walmart, Zyrtec, Neutrogena)

  • Promo Spots:

    Promos are spots that are used to promote when specific programming is airing or to provide station or Network information (CBS-Big Bang Theory+Promo; CBS-Network Info Msg+Promo, Station Info Msg+Promo). A Promo can be either a paid promo or a non-paid promo.

  • Non-Paid Promo:

    A non-paid promo are promo spots that were not 'paid for' by a Network or local station. Non-paid promos are used for Syndicated titles (Big Bang Theory+Promo) and same Network promos (CBS-Big Bang Theory+Promo on a CBS affiliated station)

  • Paid Promo:

    A paid promo is used when programming is being advertised for a non-affiliated Network or Cable station (CBS-Big Bang Theory+Paid Promo if airing on a non-CBS affiliated station or ESPN-MLB Baseball+Paid Promo when promo is airing on a station not affiliated with the Cable network ESPN)

  • PSA:

    A PSA is a Public Service Announcement spot. Local stations are not required to air Network PSA's and can choose to air local spots instead.

  • Pod:

    A pod is a group of spots airing together.

  • Flag:

    A flag is the origin indicator of a spot. Spots can be local, Network, Syndicated, Cable, or Hispanic in origin. Network, Syndication, Cable, and Hispanic Network titles have specific sets of spots that must be flagged in the local edits when those titles are airing. The flagged spots per title vary each broadcast date.

  • Recover Log:

    A stored copy of how a local edit originally came in before an editor has made any changes. Recover Logs can be brought up by using the 'Log' function at the top of the Spot EWS screen. Recover Logs are used to recover parts of an edit that were incorrectly deleted.

  • Total Errors:

    The list of Total Errors that come in with a local edit.

  • Error:

    The individual Error items found in a local edit's Total Error List. Total Errors are red flags raised by the system regarding possible problems or incorrect entries in a local edit. Each Total Error should be carefully investigated within the edit before being approved or deleted from the Queue

  • Split Screen Editing:

    The term used to describe the most accurate Spot Editing process. Using split screens allows the editor to compare a local edit with titles and spots found in a Network, Syndicated, Spot, Cable, or Hispanic Network log to correctly identify titles, lengths, and the spots that should be flagged in a local edit.

  • Validate:

    Used after editing a station or market to make sure no additional Errors are triggered before closing a market
