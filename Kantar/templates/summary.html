Introduction to Broadcast TV


What is editing?


  • The main goals of Spot TV Editing are to resolve origin discrepancies (the source of the commercial as either Network, Syndication or Local. Also known as ‘flags’), verify and correct program titles and lengths, including verifying and correcting MediaWatch errors (Misclassification of spots), report missing time (gaps in software-collected data) as needed, and verifying Network & regional sporting events (Local sporting events not airing nationally).​​
  • There are predetermined problems with the data that the Editorial Workstation (EWS, the software application used for Editing) recognizes and arranges in Queues. The editors must resolve all items in the Queues before the data can be released to the next department.

Editing Vocabulary


Broadcast date
 The Broadcast Date of an edit or episode title is when the edit or episode aired on television. Not to be confused with a calendar date. For example, on calendar date of Monday 6/19, Spot editors are working on Broadcast Date 6/14.
Market
 The specific city in the U.S. or Canada that is monitored by Kantar Media. There are currently 215 markets monitored.
Station
 Within each market, the stations are the individual networks monitored in that market.
Pod
 A pod is a group of spots airing together.
Spot
 Spot is another term for 'commercial' or 'ad'
Local Edit=Log
 The local edit is the list of program titles and spots airing on a market's station for a specific broadcast date.  The local edit is what Spot editors review and edit for accuracy.
Network edit=Log
 The Network edit provides the list of an individual Network's program titles, program lengths, and Network spots for a specific broadcast date.  It is created by the Network editors after reviewing the Network video.  Network titles can only air on that Network's local station or affiliate. 
Pacific Network Edit/log
 Pacific Network Edits provide information for West Coast/Arizona/Alaska/ Hawaii programming for a specific broadcast date.  Pacific edits are only referenced when needed.  Current Pacific Network feeds include ABCP, CBSP and NBCP.  
Cable Network Edit
 The Cable Network edit provides the list of an individual Cable Network's programming information airing on a specific Cable Network for a specific broadcast date.
Syndication Edit
 The Syndication edit provides the list of specific Syndicated spots that are airing with a Syndicated title as well as the length of the Syndicated title.  Syndicated titles can air on any Network's local station.
Hispanic Edit
 Hispanic edits provide the list of an individual Hispanic Network's program titles, program lengths, and Hispanic Network spots for a specific broadcast date.  It is created by the Hispanic Network editors after reviewing the Network video.   Current Hispanic Networks include AZTA, TEL, UMA, and UNI.
Brand spots
 Brand spots include stores and individual products (Walmart, Zyrtec, Neutrogena)
Non-Paid Promo
 A non-paid promo are promo spots that were not 'paid for' by a Network or local station.  Non-paid promos are used for Syndicated titles (Big Bang Theory+Prom)  and same Network promos (CBS-Big Bang Theory+Promo on a CBS affiliated station)
PSA
 PSA is a Public Service Announcement spot.  Local stations are not required to air Network PSA's and can choose to air local spots instead.
Flag
 A flag is the origin indicator of a spot. Network, Syndication, Cable, and Hispanic Network titles have specific sets of spots that must be flagged in the local edits when those titles are airing.  The flagged spots per title vary each broadcast date.
Recover Log
A stored copy of how a local edit originally came in before an editor has made any changes.  Recover Logs can be brought up by using the 'Log' function at the top of the Spot EWS screen.  Recover Logs are used to recover parts of an edit that were incorrectly deleted. 
Total Errors
The list of Total Errors that come in with a local edit.
Error
The individual Error items found in a local edit's Total Error List.  Total Errors are red flags raised by the system regarding possible problems or incorrect entries in a local edit.  Each Total Error should be carefully investigated within the edit before being approved or deleted from the Queue.
Split Screen Editing
Using split screens allows the editor to compare a local edit with titles and spots found in a Network, Syndicated, Spot, Cable, or Hispanic Network log to correctly identify titles, lengths, and the spots that should be flagged in a local edit.


Promos/Paid promos



Promo Spots
 Promos are spots that are used to promote when specific programming is airing or to provide station or Network information. A Promo can be either a paid promo or a non-paid promo.
Non-Paid Promo
 A non-paid promo are promo spots that were not 'paid for' by a Network or local station. Non-paid promos are used for Syndicated titles (Big Bang Theory+Promo) and same Network promos (CBS-Big Bang Theory+Promo on a CBS affiliated station)
Paid Promo
 A paid promo is used when programming is being advertised for a non-affiliated Network or Cable station (CBS-Big Bang Theory+Paid Promo if airing on a non-CBS affiliated station or ESPN-MLB Baseball+Paid Promo when promo is airing on a station not affiliated with the Cable network ESPN)
