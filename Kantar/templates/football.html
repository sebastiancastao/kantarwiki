Football Editing Guidelines

  1. Weekly Football Schedule
    • Monday
      ESPN Cable game, edited by QA team to ESPN Cable Network
    • Thursday
      NFLN Cable game, edited by QA team to NFLN Cable Network
    • Friday
      High School Football: identify lengths of games using spots in edit
    • Saturday
      College Football: use Network Sports email
    • Sunday
      NFL Football: use Network Sports email
    • Any day
      All of the above possible. Replays also possible anytime. Ask QA team as needed


  2. Network Sports Email
    • Use ONLY the information listed for the game in your edit
    • ONLY the Net team games get the Pre Kick & Post Gun titles
    • Use the exact titles shown in the email, edits may come in with wrong versions
    • Pay attention to special notes: in progress, markets with different game line-ups,etc


  3. Net Log Team Games listed on Sports email:
    • Only these games listed on the email should have the Pre Kick title
    • Spots in edits for Net games usually match the sequence in the Net log


  4. Regional Games listed on Sports email:
      5. Regional games do not get the Pre Kick title
    • Do not move regional game start time to insert Pre Kick title: that title is for Net game only
    • The Network spots for the Pre Kick title are part of the regional game instead of the separate pre kick title.
    • Do not insert the Post Gun title at end of Regional games
    • The Network spots for the Post Gun title are part of the regional game instead of the separate post gun title.
    • Network spots for gm 1 Regional games can air with gm 2 Regional games, gm 2 spots can air with gm1
    • DO NOT switch game 1 and game 2 titles in edits: these are live events, game 2 does not air before game 1


  5. Special Replay Titles
    • NFL Football game replays need specific titles depending on following guidelines:
      • *USING THE YEAR IN THE "NFL FOOTBALL" SUBTITLE:
        • If the replay is a game from current season, 2021/2022:
          • change the title 'NFL Football' to 'NFL Game Replay'
        • If the replay is a game from any other year:
          • change the title 'NFL Football' to 'NFL Football Classics'
